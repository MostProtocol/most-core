In-depth documentation on Most Protocol is available at [mostprotocol.org](https://mostprotocol.org).

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
